# 👥 KeenKeeper — Keep Your Friendships Alive

KeenKeeper is a modern friendship management web app built with **Next.js**.  
It helps users track their friends, maintain relationships, and analyze interaction history like calls, texts, and video chats.

---

## 🚀 Live Demo

👉 Live Site: https://keen-keeper-one-eta.vercel.app/  
👉 GitHub Repo: https://github.com/tishadey45/keen-keeper

---

## 🧠 Features

- 👥 Display friends from JSON data
- 📄 Detailed friend profile page
- 📞 Quick check-in system (Call / Text / Video)
- 📜 Timeline history of interactions
- 📊 Friendship analytics dashboard (Pie Chart)
- 🔍 Filter timeline by interaction type
- ⚡ Toast notifications on actions
- 📱 Fully responsive design (mobile, tablet, desktop)
- ❌ Custom 404 page for invalid routes
- ⏳ Loading state while fetching data

---

## 🛠️ Technologies Used

- ⚛️ Next.js (App Router)
- ⚛️ React.js
- 🎨 Tailwind CSS
- 📊 Recharts (Data Visualization)
- 🔔 React Toastify (Notifications)
- 🗂️ LocalStorage (for timeline history)
